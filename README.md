# passgener

## Feature rich generator

### Dependencies
**Python**
```sh
pip install tk pyperclip qrcode
```

**On debian**
```sh
sudo apt install python3-tk python3-pil.imagetk
```

**On fedora**
```sh
sudo dnf install python3-tkinter python3-pillow-tk 
```
